=========================================================================================================

How to compile (Option 1):
    Open a terminal inside the project directory and run the following command: "make"

    Note: If you want to clean the directory you can invoke "make clean"

How to compile (Option 2):
    Open a terminal inside the project directory and run the following command: "gcc rwp.c -lpthread -o rwp"

=========================================================================================================

How to run:
    The program can be ran as follows: "./rwp <#readers> <#writers>"

    Note: The number of readers/writers passed are integers. Do not include the triangle brackets.
